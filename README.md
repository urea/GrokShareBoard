# Grok Share Board

**Grok Share Board** は、xAIのGrokによって生成された画像や動画（Grok Imagine）を共有・閲覧・検索できるキュレーションプラットフォームです。
見やすいグリッドレイアウトを採用し、視覚的に楽しめるインターフェースを提供します。ユーザー登録不要・匿名で、誰でも手軽にGrokの作品を共有できます。

## 🚀 主な機能

- **Grok URL共有**: `https://grok.com/imagine/...` のリンクを貼り付けるだけの簡単投稿。
- **完全レスポンシブデザイン**: 
  - スマートフォンからワイドモニター（最大1920px）まで、あらゆるデバイスで快適に閲覧できるグリッドレイアウトを採用。
  - 画面幅をして最大限の一覧性を提供します。
- **スマートプレビュー & リトライ**:
  - Grokの「初回アクセス時に画像生成」という仕様に対応。
  - プレビュー未生成時は「手順案内」を表示し、ワンクリックで**画像の再読み込み (Retry Image)** が可能です。
- **日英バイリンガル対応**: UIは日本語と英語の併記に対応しており、海外ユーザーも利用可能です。
- **強力な検索機能**: サーバーサイド検索により、過去の全投稿からプロンプトで検索可能。
- **便利機能**:
  - **Copy Button**: プロンプト全文をワンクリックでクリップボードにコピー。
  - **匿名利用**: ユーザーIDの表示を撤廃し、完全にコンテンツ中心の設計に。

## 📝 使い方

1. **投稿する**:
   - Grokで画像/動画を生成し、そのURLをコピーします。
   - サイト上部の入力欄にペーストし、"Load / 読み込み" をクリックします。
   - プレビューを確認し、"Share / 投稿する" で完了です。
   - ※もしプレビューが表示されない場合は、画面の案内（Check Original -> Retry Image）に従ってください。
2. **閲覧・活用する**:
   - トップページに最新の投稿がグリッド表示されます。
   - 画像をクリックするとGrokの元ページが開きます。
   - プロンプトをコピーして、自分の創作に役立てることができます。

## 💻 技術スタック (Tech Stack)

このプロジェクトは、サーバーレスかつ静的ホスティング可能な構成で設計されています。

- **Framework**: Next.js 15+ (App Router)
- **Styling**: Tailwind CSS v4
- **Database**: Supabase (PostgreSQL)
- **Deployment**: GitHub Pages (Static Export) / Vercel

### 開発・運用メモ
このプロジェクトは、Grokの生成物が「実際にユーザーがそのページを訪問しないと生成されない（Lazy Generation）」仕様や、CORS制限などの課題をクライアントサイドの工夫（リトライロジック、動的フォールバック）で解決しています。
